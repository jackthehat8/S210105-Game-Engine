creates the surface and texture for the sprite

Parameters
fileNamethe address that the .bmp file is
owner_the BaseObject that the component is to be added onto
renderLayer_the layer for the sprite to be rendered on
useTransparencyif the sprite should remove the background

Note
for this to work the background should be the colour {255,255,0,0}

Example

BaseObject* tempObject = new BaseObject("temp", 550, 275, 0);
tempObject->AddComponent(new Sprite("assets/temp.bmp", tempObject, 0, true));
