Detailed Description

this is used to display text onto the screen

it can be paired with a sprite on the samae BaseObject to add a background behind the text

This loads the font that is input

Parameters
fileNamethe address that the .ttf file is
fontSizethe size the font wants to be
owner_the BaseObject that the component is to be added onto
message_the message to be displayed
colour_the colour the font should be
offset_a possible offset that can be added to centre it on a background if needed

this is defaulted to 0,0

Example

BaseObject* tempObject = new BaseObject("temp", 550, 275, 0);
tempObject->AddComponent(new Text("assets/temp.ttf", 100, tempObject, "this is a temporary message", { 0,0,0 }, { 125,50 }));
